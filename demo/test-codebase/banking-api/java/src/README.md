# Java Source Root

Root directory for Java source code of the simulated banking API (Spring Boot). Contains standard Maven project structure:

```
src/main/java/com/unionbank/api/
├── LoanController.java     — v3 loan endpoints (4 active routes)
└── ZombieController.java   — v1 admin endpoints (5 zombie routes)

src/test/                   — Test directory (empty in this mock)
```
